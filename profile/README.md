## OPCspace 是什么

OPCspace 是一个围绕 **AI 原生个体与小型组织** 建设的开放实践网络。

这里的 OPC 不只是“一人开公司”，而是一种新的组织方法：把个人判断、专业知识、业务流程和 AI Agent 组合成可以持续运行、不断复用的组织能力。

我们同时建设三类公共资产：

| OPEN SOURCE | KNOWLEDGE BASES | AI-NATIVE COMMUNITY |
|---|---|---|
| **开源工作系统** | **实践知识库** | **AI 原生组织社区** |
| 把创业、增长、合规、技术栈和应用创建封装成 Skills 与可执行工作流。 | 基于官方资料整理 Codex、WorkBuddy 等工具的中文实践路径。 | 连接正在把 AI 内化为岗位、流程、记忆和交付能力的实践者。 |
| [浏览开源项目](#开源项目--open-source) | [进入知识库](#实践知识库--knowledge-bases) | [参与共建](#参与共建--join-us) |

> **我们的目标：让 AI 从临时使用的工具，变成个人与小型组织可积累、可审计、可协作的生产能力。**

## 我们如何理解“AI 原生组织”

AI 原生不是给现有流程加一个聊天窗口，而是重新设计工作的基本单元：

```text
人的判断与责任
      ↓
可复用的知识、规则与上下文
      ↓
Agent / Skill / 自动化工作流
      ↓
可验证、可回退、可持续改进的交付
```

因此，OPCspace 关注的不只是工具推荐，还包括：

- 如何把隐性经验沉淀为组织知识；
- 如何让 Agent 理解角色、流程和验收标准；
- 如何在自动化中保留权限、审计和人工责任；
- 如何让一个人或小团队拥有过去只有大型组织才具备的执行系统。
