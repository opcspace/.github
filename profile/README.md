<summary><h2 style="display: inline-block">🇨🇳 国内版 · Domestic</h2></summary>

> **OPC Space 国内版是一个围绕 AI 一人公司（One-Person Company, OPC）建立的创业社区**，不是商业平台。我们连接有相同理念的个体创业者，共享政策信息、产业洞察、技术工具与落地经验。

### 社区提供什么

| 板块 | 说明 |
|------|------|
| 📊 [产业报告](https://opcspace.com.cn/) | 31 省市自治区、372+ 园区、14 个研究维度的开放产业研究 |
| 🏛️ [政策导航](https://opcspace.com.cn/) | 社区共建的全国政策地图，算力券 / 模型券一键查询 |
| 🎯 [路径匹配](https://opcspace.com.cn/) | 根据创业阶段、城市、行业匹配适合你的落地路径 |
| 📚 [创业课程](https://opcspace.com.cn/) | 16 课社区方法论：战略定位、合规落地、技术工具、获客运营、增长进阶 |
| 🛠️ [开源 Skill](https://github.com/opcspace) | 社区成员贡献的 Agent 执行型工作流 |
| 🤝 [资源对接](https://opcspace.com.cn/) | 园区、政策、工具、人才等社区资源互助 |

### 社区数据

- 🗺️ **31** 省市自治区
- 🏙️ **70** 已收录城市
- 🏢 **372+** 园区 / 社区
- 📋 **125+** 政策记录
- 📐 **14** 研究维度
- 👥 社区共建 · 优先展示官方原文 · 拒绝 AI 编造 · 多源交叉核实

### 适合谁加入

- 🎯 计划创业，选址迷茫 → [做路径匹配](https://opcspace.com.cn/)
- 🏛️ 已有项目，缺政策资源 → [查政策导航](https://opcspace.com.cn/)
- 📚 零基础，需要系统学习 → [看创业课程](https://opcspace.com.cn/)
- 💡 有技术/经验，愿意贡献 → [提交开源 Skill](https://github.com/opcspace)

<summary><h2 style="display: inline-block">🌍 International Version · 海外版</h2></summary>

> **OPC Space is a global community for AI-powered One-Person Companies (OPC)**. We connect solo entrepreneurs, share policy insights, industry research, tooling workflows, and landing experiences across markets.

### What the Community Offers

| Section | Description |
|---------|-------------|
| 📊 [Industry Reports](https://opcspace.com.cn/) | Open industry research covering 31 provinces, 372+ parks, and 14 research dimensions |
| 🏛️ [Policy Navigator](https://opcspace.com.cn/) | Community-built national policy map with one-click search for compute & model vouchers |
| 🎯 [Path Matching](https://opcspace.com.cn/) | Match your startup stage, city, and industry with the best landing path |
| 📚 [Startup Courses](https://opcspace.com.cn/) | 16-lesson community methodology: strategy, compliance, tech stack, growth, and operations |
| 🛠️ [Open Source Skills](https://github.com/opcspace) | Agent execution workflows contributed by community members |
| 🤝 [Resource Matching](https://opcspace.com.cn/) | Community-driven resource sharing: parks, policies, tools, and talent |

### Community Highlights

- 🗺️ **31** provinces covered
- 🏙️ **70** cities indexed
- 🏢 **372+** parks & communities
- 📋 **125+** policy records
- 📐 **14** research dimensions
- 👥 Community-built · Official-first sources · No AI hallucination · Multi-source verification

### Who It's For

- 🎯 Planning to start up but unsure where to land → [Path Matching](https://opcspace.com.cn/)
- 🏛️ Have a project but need policy resources → [Policy Navigator](https://opcspace.com.cn/)
- 📚 Starting from zero and need a structured methodology → [Courses](https://opcspace.com.cn/)
- 💡 Have skills/experience and want to contribute → [Submit Open Source Skills](https://github.com/opcspace)

</details>
